# Introduction to Generative AI and Machine Learning (Fall 2025)

**Course Name (Chinese):** 生成式人工智能与机器学习导论2025秋

## Course Description

This folder contains learning materials, assignments, and experiments for the Introduction to Generative AI and Machine Learning course offered in Fall 2025.

## Contents

- Jupyter notebooks (.ipynb) for labs and assignments
- Course materials and notes
- Project files

## Getting Started

Make sure you have Python and Jupyter Notebook installed. You can install the required packages using:

```bash
pip install jupyter numpy pandas matplotlib scikit-learn
```

To start Jupyter Notebook:

```bash
jupyter notebook
```
