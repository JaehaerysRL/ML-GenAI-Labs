## File Structure

Each course folder contains:

## Getting Started

### Prerequisites

- Python 3.x
- Jupyter Notebook
- Common ML libraries (numpy, pandas, matplotlib, scikit-learn)

### Installation

```bash
pip install jupyter numpy pandas matplotlib scikit-learn
```

### Running Jupyter Notebooks

```bash
jupyter notebook
```
