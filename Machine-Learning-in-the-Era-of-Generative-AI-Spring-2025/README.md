# Machine Learning in the Era of Generative AI (Spring 2025)

**Course Name (Chinese):** 生成式AI时代下的机器学习2025春

## Course Description

This folder contains learning materials, assignments, and experiments for the Machine Learning in the Era of Generative AI course offered in Spring 2025.

## Contents

- Jupyter notebooks (.ipynb) for labs and assignments
- Course materials and notes
- Project files

## Getting Started

Make sure you have Python and Jupyter Notebook installed. You can install the required packages using:

```bash
pip install jupyter numpy pandas matplotlib scikit-learn
```

To start Jupyter Notebook:

```bash
jupyter notebook
```
