# Machine Learning (Spring 2021)

**Course Name (Chinese):** 机器学习2021春

## Course Description

This folder contains learning materials, assignments, and experiments for the Machine Learning course offered in Spring 2021.

## Contents

- Jupyter notebooks (.ipynb) for labs and assignments
- Course materials and notes
- Project files

## Getting Started

Make sure you have Python and Jupyter Notebook installed. You can install the required packages using:

```bash
pip install jupyter numpy pandas matplotlib scikit-learn
```

To start Jupyter Notebook:

```bash
jupyter notebook
```
